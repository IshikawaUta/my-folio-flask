Proyek Saya

Tentang Saya

Sebagai seorang web developer profesional yang bersemangat, saya berdedikasi untuk menciptakan solusi digital yang inovatif dan berfokus pada pengguna. Saya memiliki keahlian dalam pengembangan "front-end" (dengan teknologi modern seperti HTML5, CSS3, JavaScript, dan framework seperti React atau Tailwind) dan "back-end" (menggunakan Python dengan framework Flask, atau teknologi relevan lainnya). Kombinasi keterampilan ini memungkinkan saya untuk membangun "aplikasi web" yang kuat, dinamis, responsif, dan memberikan pengalaman pengguna yang mulus dari awal hingga akhir.
